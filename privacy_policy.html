中文版

1. 概述

感谢您使用扫雷游戏（以下简称"本应用"）。我们非常重视您的隐私保护。本隐私政策旨在帮助您了解本应用如何处理信息。

2. 信息收集

本应用本身不收集、存储或传输任何用户个人信息。

本应用是一款单机游戏，所有游戏数据（包括游戏进度、统计数据、设置偏好等）仅存储在您的设备本地，不会上传到我们的服务器。

2.1 广告服务

本应用集成了第三方广告服务以支持应用的持续开发和维护。广告服务提供商可能会收集以下信息：

  • 设备信息：设备型号、操作系统版本、唯一设备标识符（如广告ID）
  • 使用信息：应用使用情况、广告展示和点击数据
  • 位置信息：大致地理位置（基于IP地址，用于展示相关广告）

这些信息由广告服务提供商收集和处理，用于提供个性化广告和分析广告效果。我们不会直接访问或存储这些信息。

3. 数据存储

本应用使用Android系统的本地存储功能来保存以下数据：

  • 游戏统计数据：游戏次数、胜率、最佳成绩等
  • 用户设置：难度选择、是否显示过新手引导等偏好设置

这些数据仅存储在您的设备上，不会被传输或分享给任何第三方。

4. 权限说明

本应用可能请求以下Android系统权限：

4.1 存储权限（可选）

  • 用途：保存游戏统计数据和用户设置到设备本地存储
  • 说明：这些数据仅存储在您的设备上，不会上传到网络

4.2 震动权限（如果实现了震动功能）

  • 权限名称：VIBRATE
  • 用途：在特定游戏事件（如点击地雷）时提供触觉反馈
  • 说明：此权限仅用于增强游戏体验，不涉及任何数据收集

重要说明：本应用不会请求网络权限、位置权限、相机权限、麦克风权限或联系人权限等敏感权限。

5. 第三方服务

本应用不集成任何第三方SDK、分析工具、广告服务或社交媒体插件。

6. 儿童隐私

本应用适合所有年龄段用户使用，不会主动收集任何用户（包括儿童）的个人信息。

7. 数据安全

由于本应用不收集或传输任何用户数据，因此不存在数据泄露的风险。所有游戏数据仅存储在您的设备本地，受您设备的安全措施保护。

8. 隐私政策变更

我们可能会不定期更新本隐私政策。任何变更将通过应用更新的方式通知您。建议您定期查看本隐私政策以了解最新信息。

9. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

邮箱：minesweeper.game.support@example.com

English Version

1. Overview

Thank you for using Minesweeper Game (hereinafter referred to as "the App"). We take your privacy very seriously. This Privacy Policy is designed to help you understand how the App handles information.

2. Information Collection

The App does NOT collect, store, or transmit any personal user information.

The App is a completely offline single-player game. All game data (including game progress, statistics, settings preferences, etc.) is stored only locally on your device and is not uploaded to any server or third-party service.

3. Data Storage

The App uses Android's local storage features to save the following data:

  • Game Statistics: Number of games played, win rate, best scores, etc.
  • User Settings: Difficulty selection, tutorial display preferences, etc.

This data is stored only on your device and is not transmitted or shared with any third parties.

4. Permissions

The App may request the following Android system permissions:

4.1 Storage Permission (Optional)

  • Purpose: Save game statistics and user settings to local device storage
  • Note: This data is stored only on your device and is not uploaded to the network

4.2 Vibration Permission (If vibration feature is implemented)

  • Permission Name: VIBRATE
  • Purpose: Provide haptic feedback during specific game events (such as clicking on a mine)
  • Note: This permission is only used to enhance the gaming experience and does not involve any data collection

Important Note: The App does not request sensitive permissions such as network access, location, camera, microphone, or contacts.

5. Third-Party Services

The App does not integrate any third-party SDKs, analytics tools, advertising services, or social media plugins.

6. Children's Privacy

The App is suitable for users of all ages and does not actively collect personal information from any users (including children).

7. Data Security

Since the App does not collect or transmit any user data, there is no risk of data leakage. All game data is stored locally on your device and is protected by your device's security measures.

8. Changes to Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be communicated to you through app updates. We recommend that you review this Privacy Policy regularly to stay informed of the latest information.

9. Contact Us

If you have any questions or suggestions about this Privacy Policy, please contact us:

Email: minesweeper.game.support@example.com
